## 🎮 Как пользоваться приложением

### Для организатора:

1. **Регистрация** → выбери роль "Организатор"
2. **Создать квиз** → нажми "Создать квиз" в меню
3. **Заполни настройки**: название, категория, время на вопрос
4. **Добавь вопросы**: нажми "Добавить вопрос", введи текст, варианты ответов, отметь правильный (нажав на букву), сохрани
5. **Запусти квиз**: нажми " Запустить"
6. **Поделись кодом комнаты** — участники видят его на экране (6 букв/цифр)
7. Нажми **"Начать квиз"** когда все подключились
8. **Управляй вопросами**: нажимай "Показать ответы" → "Следующий вопрос"

### Для участника:

1. **Регистрация** → выбери роль "Участник"
2. Нажми **"Войти в игру"** в меню
3. Введи **код комнаты** от организатора
4. Жди начала квиза
5. Выбирай ответы и нажимай **"Подтвердить"**
6. После каждого вопроса видишь промежуточный рейтинг
